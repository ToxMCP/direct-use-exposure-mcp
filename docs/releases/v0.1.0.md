# Direct-Use Exposure MCP v0.1.0

Direct-Use Exposure MCP `v0.1.0` is the first public deterministic external-dose release candidate for the ToxMCP suite.

## Included In This Release

- Dermal, direct-use/incidental oral, and inhalation screening scenario construction
- Aggregate/co-use summaries and auditable scenario comparison
- PBPK handoff export aligned to the published PBPK MCP request shape
- Deterministic ToxClaw evidence and refinement-bundle exports

## Benchmark Summary

- Benchmark cases: `22`
- Covered case IDs: `dermal_hand_cream_screening`, `inhalation_trigger_spray_screening`, `inhalation_air_space_insecticide_aerosol_screening`, `inhalation_air_space_insecticide_aerosol_time_series_0p75h_2001`, `inhalation_air_space_insecticide_aerosol_time_series_6h_2001`, `inhalation_residual_air_reentry_chlorpyrifos_screening`, `inhalation_residual_air_reentry_chlorpyrifos_time_series_1990`, `inhalation_residual_air_reentry_diazinon_time_series_1990`, `inhalation_tier1_trigger_spray_nf_ff`, `inhalation_tier1_disinfectant_trigger_spray_external_2015`, `oral_direct_oral_screening`, `oral_medicinal_liquid_delivered_dose_screening`, `dermal_density_precedence_volume_case`, `cross_route_aggregate_summary`, `zero_baseline_comparison`, `dermal_pbpk_export`, `dermal_pbpk_external_import_package`, `inhalation_tier1_scenario_package_probability`, `worker_inhalation_janitorial_trigger_spray_execution`, `worker_inhalation_handheld_biocidal_trigger_spray_execution`, `worker_dermal_wet_wipe_gloved_hands_execution`, `worker_dermal_handheld_biocidal_trigger_spray_execution`

- `inhalation_trigger_spray_screening` [scenario] Adult EU trigger spray inhalation case with explicit room volume and regional default ventilation/duration.
- `inhalation_air_space_insecticide_aerosol_screening` [scenario] Adult EU air-space insecticide aerosol case sized to a narrow household mosquito aerosol indoor-air benchmark band.
- `inhalation_air_space_insecticide_aerosol_time_series_0p75h_2001` [scenario] Adult air-space insecticide aerosol case aligned to the 0.75-hour room-air concentration point from the household mosquito aerosol study.

- `dermal_density_precedence_volume_case` [scenario] Adult dermal cream case covering physical-form density override precedence for mL inputs.
- `cross_route_aggregate_summary` [aggregate] Aggregate summary spanning the canonical dermal and inhalation benchmark scenarios.

- `worker_inhalation_janitorial_trigger_spray_execution` [worker_inhalation_execution] Canonical worker inhalation surrogate execution case for a janitorial disinfectant trigger spray with explicit control and respiratory-protection modifiers.
- `worker_inhalation_handheld_biocidal_trigger_spray_execution` [worker_inhalation_execution] Study-like small-scale handheld BAC trigger-spray worker case aligned to the 2023 occupational biocidal spray monitoring paper.
